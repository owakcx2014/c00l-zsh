# 🚀 c00l-zsh: The Ultimate ZSH Experience

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-Linux-red.svg)
![Year](https://img.shields.io/badge/year-2026-gold.svg)

**c00l-zsh** هو سكربت أتمتة (Automation Script) مصمم خصيصاً لمستخدمي **Linux** لتحويل التيرمينال الممل إلى تجربة بصرية وإنتاجية خارقة في ثوانٍ معدودة.

## ✨ المميزات (The Butter Features)
- 🏎️ **تثبيت صامت:** يقوم بتثبيت Zsh و Oh My Zsh وجميع الإضافات الضرورية بضغطة زر.
- 🎨 **ثيمات احترافية:** إعداد تلقائي لثيمات Powerlevel10k مع واجهة zsh الأسطورية.
- ⚡ **إضافات السرعة:** تفعيل `zsh-autosuggestions`*(الاقتراحات التلقائية)* و `zsh-syntax-highlighting` *(تلوين النص)* تلقائياً.
- 🛡️ **نظيف وآمن:** سكربت مكتوب بلغة Bash واضحة، لا "عك" ولا تعقيد.

## 🚀 طريقة التثبيت السريع (One-Liner)
افتح التيرمينال الخاص بك والصق الأمر التالي:

```bash
git clone https://github.com/owakcx2014/c00l-zsh.git
cd c00l-zsh
chmod +x install.sh
./install.sh
```
اختر نوع نظام التشغيل arch/Debian/Fedora وأدخل كلمة المرور واضغط y اذا سألك،
بعد التثبيت ستجد ملف اسمه `zsh shortcuts.txt` في مجلد المنزل (~) هكذا `~/zsh shortcuts.txt` اقرأه واستمتع بالـZSh الرائع!!
